BUS Core

Local-first ERP for small workshops

Tracks inventory and manufacturing without the cloud.

Download Latest Version View Changelog

What It Does

  • FIFO inventory tracking
  • Manufacturing recipes and production runs
  • Real gross profit via inventory-linked COGS
  • Local cash event tracking
  • Encrypted database export
  • REST API for integrations

What It Does Not Do

  • No forced user accounts
  • No background telemetry
  • No subscription billing
  • No cloud sync or remote access
  • No remote kill switch

Philosophy

Software should not own your business. BUS Core is opinionated: inventory is ledger-driven, cost is derived from truth, and accounting bloat stays out. It is designed for a 1–10 person shop that prefers control over convenience.

Technical Overview

Backend:SQLite (WAL mode)
API:FastAPI, local-only bind

Trust and Security

BUS Core — the digital shop jig for manufacturing workflows.

Local-First Architecture

BUS Core runs on your machine. Database: SQLite. No required external services.

No Telemetry

No background analytics. No usage tracking. No hidden calls home.

Network Surface

Default bind: 127.0.0.1:8765. Not publicly exposed unless explicitly configured.

Data Ownership

Your database resides in your user directory. You control backups and exports.

Backups

Encrypted export via AES-GCM. Portable, self-contained database files.

License

Open Source under AGPL. Inspected. Auditable. Self-hostable. Binaries are digitally signed.
